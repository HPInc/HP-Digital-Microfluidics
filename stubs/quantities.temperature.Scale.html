quantities.temperature.Scale

class quantities.temperature.Scale(abbr: str, absolute_zero: float, relative_to_kelvin: float)

Bases: object

A unit to be used to specify a TemperaturePoint.

To obtain a TemperaturePoint, multiply a float by a Scale, e.g.,

tp = 25*abs_C
