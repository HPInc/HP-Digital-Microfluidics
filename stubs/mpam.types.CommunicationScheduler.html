schedule_communication(cb: Callable[[], Optional[Callable[[], Any]]], *, after: Optional[Union[mpam.types.Ticks, quantities.dimensions.Time]] = None) → None

Schedule communication of cb after optional delay after

This is typically implemented by delegating and will result in calling SystemComponent.schedule(), which will call System.on_tick() if after is a Ticks value and System.communicate() if after is a Time value. If after is None, it is up to the SystemComponent to determine whether it should be interpreted as zero ticks or zero seconds.

Parameters

cb – the callback function to schedule.

Keyword Arguments

after – An optional delay before scheduling.
