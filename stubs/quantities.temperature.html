class quantities.temperature.TemperaturePoint(absolute: quantities.dimensions.Temperature)

Bases: object

An absolute temperature. For temperature differences use Temperature.

TemperaturePoints are most often created by multiplying a number and a Scale, such as abs_C, abs_K, or abs_F, e.g.,

tp = 94.2*deg_C

Note

abs_K is a Scale for TemperaturePoints. SI.deg_K is a Unit[Temperature].

If tp is a TemperaturePoint and t is a Temperature, then

tp2 = tp + t
tp2 = t + tp
tp2 = tp - t
t = tp - tp2

are all valid statements.

TemperaturePoints may be compared, regardless of the Scales used to create them:

if 25*abs_C < 60*abs_F:
    ...

A TemperaturePoint can be converted to a string (on_scale()) or a float (as_number()) by specifying Scale. If a TemperaturePoint is converted to a string by using str(), on_scale() is called using default_scale, which defaults to deg_C, but can be changed.
