mpam.types.Reagent

class mpam.types.Reagent(name: str, *, composition: Optional[Mapping[mpam.types.Chemical, Union[quantities.dimensions.MolarConcentration, quantities.dimensions.Density, quantities.dimensions.VolumeConcentration, mpam.types.UnknownConcentration]]] = None, min_storage_temp: Optional[quantities.temperature.TemperaturePoint] = None, max_storage_temp: Optional[quantities.temperature.TemperaturePoint] = None)

Bases: object

A reagent. All Reagents have

  • a name

  • a chemical composition (which will be an empty dict unless specified) as a mapping from Chemical to Composition

  • optional minimum (min_storage_temp) and maximum (max_storage_temp) storage temperatures

  • a mixture description, useful when a Reagent is a not- otherwise-named mixture of two or more Reagents

  • an indication of whether the Reagent is_pure (i.e., not a mixture).

  • a tuple of process_step that the Reagent has gone through.

  • the unprocessed Reagent prior to any processing. (Often this Reagent itself.)

Reagents can be ordered, and sort by their names.

property mixture: Tuple[tuple, ...]

A description of the Reagent as a mixture of other Reagents. The description is a tuple of tuples, each of which contains a Reagent and a Fraction specifying how much of the overall mixture that Reagent constitutes.

The elements of this tuple will be sorted by Reagent name, rendering the whole thing useful as a mapping key.

This Reagent is_pure() if and only if the the resulting tuple has one element, and the first element of that element will be this Reagent.

The second elements of each element of this tuple (Fractions) will all add up to 1.

A new version of the Reagent after it has been processed.

If this Reagent is the waste_reagent, it is returned unchanged.

The results of calling processed() on a given Reagent are cached, and an existing value will be reused.

If a new one is created, the composition and minimum and maximum storage temperatures are modified as specified by the parameters or, if unspecified, copied from this Reagent.

Note

If a cached result of a prior call to processed() with this step is found, it will be returned, and the other arguments will be ignored.

Parameters

  • step – the ProcessStep applied or the name used to find or create it.

  • new_composition_function – an optional function to determine the new composition.

Keyword Arguments

  • min_storage_temp – an optional minimum storage temperature

  • max_storage_temp – an optional maximum storage temperature

Returns

the resulting Reagent
