mpam.types.ProcessStep

class mpam.types.ProcessStep(description: str)

Bases: object

A processing step that a Reagent can go through.

The basic notion is that if r is a Reagent and ``ps` is a ProcessStep, after

r2 = r.processed(ps)

r2 will be a different reagent with the property that r2.unprocessed() will be r (or, at least r.unprocessed()) and r2.process_steps() will end with ps.

ProcessSteps can be obtained by calling find_or_create(), passing in a description:

ps = ProcessStep.find_or_create("thermocycle")
