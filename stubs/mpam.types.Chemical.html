mpam.types.Chemical

class mpam.types.Chemical(name: str, *, formula: Optional[str] = None, description: Optional[str] = None)

Bases: object

A chemical that may exist as a component of a Reagent. It has a name and optionally a formula and a description, all strings.

The class maintains a dictionary of instances keyed by name, so the encouraged way to obtain an object is to look it up using find(), which will create the Chemical if one is not already there:

c = Chemical.find("oxygen", formula="O2")

name is final, but formula and description may be modified.

classmethod find(name: str, *, formula: Optional[str] = None, description: Optional[str] = None) → Chemical

Find a Chemical with the given name, otherwise create one. If a new Chemical is created, it is added to known.

Note

If a Chemical with the given name is already known, the formula and description will not be modified.
