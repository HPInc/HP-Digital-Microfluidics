mpam.device.DropLoc

class mpam.device.DropLoc(*args, **kwargs)

Bases: abc.ABC, mpam.types.CommunicationScheduler

A location that can hold a Drop and participate in a Blob.

It maintains a ChangeCallbackList for its drop attribute to which callbacks can be added by means of on_drop_change.

checked_drop is the same as drop, but it is guaranteed to be a Drop, raising a TypeError if drop is None.

Note

DropLoc is an abstract class. In addition to the abstract methods inherited from CommunicationScheduler, concrete subclasses must also define compute_neighbors_for_blob().
