class quantities.dimensions.Time(mag: float, dim: Optional[Dimensionality[ND]] = None)

Bases: BaseDim

A Quantity representing time.

Time objects can be converted to (objects that convert to) strings in common formats by means of in_HMS(), in_HM(), and in_MS().

A float divided by a Time yields a Frequency

SI Units include:

  • seconds (s, second, secs, sec)

  • milliseconds (ms, millisecond, millisecs, millisec, msecs, msec)

  • microseconds (µs, us, microsecond, microsecs, microsec, µsecs, µsec, usecs, usec)

  • nanoseconds (ns, nanosecond, nanosecs, nanosec, nsecs, nsec)

  • minutes, (minute, mins). * Note: min is a reserved word in Python.

  • hours (hr, hour)

  • days, (day)

  • weeks (wk, week)

  • There is no predefined Unit for months or years, as those vary in length.

in_HM(sep: str = ':') → Joined

Format the Time as hours and minutes, separated by sep. If the resulting object is formatted, the precision will only apply to the minutes. Both numbers format with (at least) two digits with leading zeroes.

Keyword Arguments

sep – the separator (default=”:”) to use between numbers

in_HMS(sep: str = ':') → Joined

Format the Time as hours, minutes, and seconds, separated by sep. If the resulting object is formatted, the precision will only apply to the seconds. All numbers format with (at least) two digits with leading zeroes.

Keyword Arguments

sep – the separator (default=”:”) to use between numbers

in_MS(sep: str = ':') → Joined

Format the Time as minutes and seconds, separated by sep. If the resulting object is formatted, the precision will only apply to the seconds. Both numbers format with (at least) two digits with leading zeroes.

Keyword Arguments

sep – the separator (default=”:”) to use between numbers
