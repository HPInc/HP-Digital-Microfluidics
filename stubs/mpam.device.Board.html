ambient_threshold: Union[quantities.temperature.TemperaturePoint, quantities.dimensions.Temperature, tuple] = Temperature[9.0]

The value used by is_ambient() to compute the threshold (based on ambient_temperature) to be used by is_ambient(). If it is a TemperaturePoint, it is used directly as the threshold. If it is a Temperature, it is added to the ambient temperature. If it is a tuple, the two values are taken to be the lower and upper bounds, respectively, with a lower Temperature subtracted from ambient_temperature.

join_system(system: mpam.device.System) → None

Join system. After this call, system will be valid and joined_system will return True.

This calls component_joined() on system.

Note

join_system() will be called for a Board during the initialization of the System. For other SystemComponents, it must be called explicitly. If a class overrides join_system(), it should delegate to the base implementation.
