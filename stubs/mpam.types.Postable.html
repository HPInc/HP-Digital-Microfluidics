mpam.types.Postable

class mpam.types.Postable

Bases: mpam.types.Delayed[mpam.types.Tcontra]

A concrete subclass of Delayed that provides a post() method to assert a value, which will be passed to any waiting callbacks.

Delayed is covariant in its type parameter, so a Delayed[Derived] can be treated as a Delayed[Base]. Conversely, the concrete subclass Postable is contravariant in its type parameter, so a Postable[Base] can be treated as a Postable[Derived].

More concretely, given

class Top: ...
class Middle(Top): ...
class Bottom(Middle): ...

p = Postable[Middle]()

p can be treated as a Delayed[Middle], a Delayed[Top], or a Postable[Bottom].

post() may only be called once (although it need not ever be called). When a value is posted, any registered callbacks will be invoked and passed the posted value.

post(val: mpam.types.Tcontra) → None

Assert a value and pass it to any registered callbacks. Any callbacks registered after this point will immediately execute with this value.

Note

post() may only be called once for any Postable object. Calling it a second time will result in an assertion failure.

Note

Once post() has been called, any registered callbacks are forgotten.
