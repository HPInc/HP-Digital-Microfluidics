class mpam.types.CombinedStaticOperation(first: mpam.types.StaticOperation[mpam.types.V], second: Union[mpam.types.Operation[mpam.types.V, mpam.types.V2], mpam.types.StaticOperation[mpam.types.V2], Callable[[], mpam.types.Operation[mpam.types.V, mpam.types.V2]], Callable[[], mpam.types.StaticOperation[mpam.types.V2]]], *, after: Optional[Union[mpam.types.Ticks, quantities.dimensions.Time]] = None)

Bases: Generic[mpam.types.V, mpam.types.V2], mpam.types.StaticOperation[mpam.types.V2]

A StaticOperation representing chaining a StaticOperation and an Operation together.

Objects of this class are built using StaticOperation.then(), StaticOperation.then_call(), StaticOperation.then_compute(), and StaticOperation.then_process(),

Parameters

  • V – the type of the value produced by the first operation

  • V2 – the type of the value produced by the second operation (and the CombinedStaticOperation overall)

Attributes

first

The first StaticOperation

second

The second Operation (or StaticOperation) or a Callable that produces one.

after

An optional delay to use between first and second
