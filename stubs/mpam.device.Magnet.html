class mpam.device.Magnet(board: Board, *, state: State[OnOff], pads: Sequence[Pad])ÔÉÅ

Bases: BinaryComponent[Magnet]

A magnet that affects a set of Pads.

As a BinaryComponent, it has a board and a current_state, it can act as both a CommunicationScheduler and a OpScheduler[Magnet], and it supports on_state_change().

To initialize this asspect of the Magnet, when it is created, it must be associated with a State[OnOff] object that knows how to effect any changes on a physical device (via its realize_state() method). If nothing needs to be done, a DummyState may be used.
