pipettor_added(reagent: Reagent, volume: Volume, *, last: bool, mix_result: Optional[Union[Reagent, str]]) → None

Called when the Pipettor has finished adding Liquid. If mix_result is not None, it should be used as the result of mixing the delivered Liquid to what is already there. If last is False, there will be at least one more transfer before the overall transfer operation is complete.

Note

This is an abstract attribute which must be overridden by concrete subclasses.

Parameters

  • reagent – the Reagent added

  • volume – the Volume of Reagent added

Keyword Arguments

  • mix_result – the (optional) result of mixing the added Liquid with what’s already there.

  • last – True if this is the last transfer in a transfer request.
