mpam.types.State

class mpam.types.State(*, initial_state: Union[mpam.types.Missing, mpam.types.T])

Bases: Generic[mpam.types.T], abc.ABC

An object that encapsulates a value and a ChangeCallbackList for that value.

State is an abstract class that is required to define realize_state(). This has to do with the way State was used in the initial system as a base class for proxies for physical devices that had to not only track state changes but be able to effect them in the real world. If this is inapplicable, use the subclass DummyState, which implements realize_state() to do nothing
