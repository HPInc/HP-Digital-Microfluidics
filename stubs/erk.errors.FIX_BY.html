erk.errors.FIX_BY

class erk.errors.FIX_BY(function: Callable[[], None])

Bases: ErrorHandler

An ErrorHandler that fixes the problem by calling an arbitrary function.

It is initialized with a factory function taking no arguments. Any value returned is ignored.

This class reimplements expect_true() and expect_false() to never evaluate their msg_fn parameters.

Note

The handler is assumed to have fixed the problem. The condition is not rechecked.

Example

s = obj.lookup(n, on_out_of_range = FIX_BY(lambda: obj.grow_to(n))
