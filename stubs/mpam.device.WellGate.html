mpam.device.WellGate

class mpam.device.WellGate(board: Board, exit_pad: Pad, exit_dir: Dir, state: State[OnOff], *, neighbors: tuple[int, ...], live: bool = True)

Bases: WellPad, LocatedPad

A WellPad that is its well’s gate.

As a LocatedPad, it has a location, which is computed based on its well’s exit_pad.

Attributes

is_gate

True because this is well's gate.

__init__(board: Board, exit_pad: Pad, exit_dir: Dir, state: State[OnOff], *, neighbors: tuple[int, ...], live: bool = True) → None

Initialize the object.

location is computed based on exit_pad and exit_dir.

Parameters

  • board – the containing Board

  • exit_pad – the Pad that will be well’s :attr:~Well.exit_pad`

  • exit_dir – the direction from this WellGate to exit_pad

  • state – the associated State[OnOff]

Keyword Arguments

  • live – is the WellPad live?

  • neighbors – the index values of WellPads that should be included in neighbors_for_blob.

compute_neighbors_for_blob() → Sequence[DropLoc]

The list of neighboring DropLocs that could be part of the same Blob. This is called the first time neighbors_for_blob is referenced, and the value is cached.

This will be the WellPads corresponding to the neighbors indices provided to __init__() plus well’s exit_pad.

Returns

the Sequence of neighbors
