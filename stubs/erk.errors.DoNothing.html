erk.errors.DoNothing

class erk.errors.DoNothing

Bases: erk.errors.ErrorHandler

An ErrorHandler that does nothing.

The condition is still evaluated, but it isn’t checked and the message is never computed. This is typically used via its instance IGNORE

Example

s = obj.lookup(n, on_out_of_range = IGNORE)
