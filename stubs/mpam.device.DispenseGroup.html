A group of Wells whose shared_pads are mirrored, so that changing the state of one changes the state of all.

Attributes

A key object (typically a string, number, or Well) used when converting to a string

A Lock used by WellMotions to wrap changes to motion and state

The in-progress WellMotion, if any for the Wells in this group.

The current WellState for the Wells in this group
