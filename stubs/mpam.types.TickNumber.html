mpam.types.TickNumber

class mpam.types.TickNumber(tick: mpam.types.Ticks)

Bases: object

An absolute tick number (as opposed to a number of Ticks).

If tn is a TickNumber and t is a Ticks, then

tn = TickNumber.ZERO()
tn2 = tn + t
tn2 = t + tn
tn2 = tn - t
t2 = tn1 - tn2
tn1 == tn2 # and <, >=, etc.

are all valid.
