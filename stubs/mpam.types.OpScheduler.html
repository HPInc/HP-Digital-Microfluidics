mpam.types.OpScheduler

class mpam.types.OpScheduler

Bases: Generic[mpam.types.CS]

A mixin base class giving a class the ability to schedule Operation[CS, V]s.

Typically, this will be class CS itself, as in:

class Drop(OpScheduler['Drop']):
    ...

There are several Operations (e.g., WaitAt, Reach, WaitFor), that are defined for all OpSchedulers.

class WaitFor(waitable: Union[mpam.types.Ticks, quantities.dimensions.Time, mpam.types.Trigger, mpam.types.Delayed[Any]])

Bases: mpam.types.Operation[mpam.types.CS, mpam.types.CS]

An Operation during which the CS object waits for a WaitableType to be satisfied.

  • If waitable is a Time or Ticks, the operation completes after that delay.

  • If waitable is a Delayed, the operation completes when a value is posted to waitable.

  • If waitable is a Trigger, the operation completes when waitable fires.

The value posted to the Delayed value returned by schedule_for() is the object for which the operation was scheduled.
