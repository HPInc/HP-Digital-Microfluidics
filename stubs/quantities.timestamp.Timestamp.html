quantities.timestamp.Timestamp

class quantities.timestamp.Timestamp(time: quantities.dimensions.Time)

Bases: object

An object representing a fixed point in time.

Its time represents the Time since the epoch used by the :module:`time` module. Typically, this will not be interesting.

If ts is a Timestamp and t is a Time, then

ts2 = ts + t
ts2 = t + ts
ts2 = ts - t
t = ts - ts2

if ts == ts2: ...
if ts < ts2: ...

are all valid statements.

The current time can be obtained by Timestamp.now(). The epoch (a Timestamp with time equal to zero) is Timestamp.epoch() (also known as Timestamp.never()). This is the same epoch that is used by the standard :module:`time` module.

The following global functions are also useful:

  • time_now() is an alias for Timestamp.now().

  • time_in() is now() plus a Time delta.

  • time_since() is the Time since another timestamp.

To format a Timestamp, use strftime(). This takes an optional format (which defaults to "%Y-%m-%d_%H:%M:%S.%f" and an optional precision, expressed as a Time Unit. The default precision is ms
