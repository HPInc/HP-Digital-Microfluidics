mpam.types.Barrier

class mpam.types.Barrier(required: int, *, name: Optional[str] = None)

Bases: Trigger, Generic[T]

A Trigger that fires when a certain number of T objects reach it. Objects reaching the Barrier can optionally “pause” there by passing in a Delayed object to which the waiting object is posted when the last object reaches the Barrier.

A Barrier may be reset (by calling reset()), optionally changing the number of objects required required to reach it. When that happens, any objects that had already reached it (and any other actions added to it as a Trigger) are forgotten.

Parameters

T – The type of object the Barrier is waiting for.
